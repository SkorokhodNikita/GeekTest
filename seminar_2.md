# Создание справки по системе контроля версий GIT 

## Как начать
**Чтобы начать работать с системой контроля версий GIT необходимо выполнить следующую команду:**

```cs
git init
```

## Как добавить файл в отслеживание

**команда:**

```
git add file
```
```
git add .
```
